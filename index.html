Media Query Demo

Ekran o'lchamiga qarab fon rangi o'zgaradi

Hozirgi ekran o'lchami: 0px

Fon ranglari:

Mobil (< 600px): Qizil
Tablet (600px - 899px): Ko'k
Desktop (900px+): Yashil

Media Query breakpoints:

  • @media (min-width: 600px) - Tablet va undan katta
  • @media (min-width: 900px) - Desktop va undan katta
  • @media (min-width: 1200px) - Katta desktop
  • @media (max-width: 599px) - Mobil (kichik ekran)
  • @media print - Chop etish uchun

Qo'llanma:

Brauzer oynasining o'lchamini o'zgartiring va fon rangi qanday o'zgarishini kuzating:

  1. Oynani toraytiring - qizil rang paydo bo'ladi (mobil)
  2. Oynani kengaytiring (600px dan katta) - ko'k rangga o'zgaradi (tablet)
  3. 900px dan katta qiling - yashil rangga o'zgaradi (desktop)
